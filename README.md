# 若杉行政書士事務所 お客様サポートアプリ

行政書士に依頼しているお客様が、必要書類・進捗・よくある質問をスマホから簡単に確認できるWebアプリです。

## 📱 アプリの構成

| ファイル | 用途 |
|----------|------|
| `index.html` | お客様側アプリ（スマホで使用） |
| `admin.html` | 行政書士・事務スタッフ向け管理画面 |

## 🚀 使い方

### お客様側アプリ
1. `index.html` をブラウザで開く
2. 担当者から伝えられた「案件番号」（例：WK-001）を入力
3. ※ デモ体験ボタンからサンプルデータで試すこともできます

### 管理画面
1. `admin.html` をブラウザで開く
2. パスワード `wakasugi2024` を入力してログイン
3. お客様の追加・進捗更新・FAQ管理などが行えます

## 🎨 デザインコンセプト

- 50代以上の方でも使いやすい大きな文字・ボタン
- 紺・白・薄い青を基調とした信頼感のあるデザイン
- 専門用語をやさしい言葉で補足

## ⚙️ 技術仕様

- 純粋なHTML/CSS/JavaScript（フレームワーク不使用）
- データはブラウザの `localStorage` に保存
- サーバー不要・インストール不要
